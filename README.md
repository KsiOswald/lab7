Определить разность суммы элементов первой и второй половины файла. 



### Алгоритм решения



Родительский класс содержит строковое поле, и 3 конструктора (по умолчанию, по значению и копирование), а также одно свойство ( выдача и присваивание с проверкой на равенство нулевой строке)
Также содержит метод, который на выход отдает первый и последний символы строки и переопределение метода ToString

Дочерний класс имеет новые методы - эффектиность сокращения("100-(2*100/Длину исходной строки)% 
И разворот строки - сложение 2-х символов в строку - последнего и первого.
