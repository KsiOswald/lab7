## Задание 2



### Текст задачи

Вычислить сумму элементов. 


### Алгоритм решения


Создается файл с несколькими числами в строке. Создается случайным образом количество строк, затем количество элементов в строке, после сами числа в строке.
Сумма чисел в файле находится путем проверки следущего символа, если символ является числом - то он прибавляется к строке, который накапливает в себе цифры числа, если следующий символ - это пробел - то новое число прибавляется к сумме
Если следующий символ - это символ конец строки "\r", то считывается 2 следующих символа, тк переход на новую строку представляет собой сочетание символов "\r\n"
Сумма выводится на экран

## Задание 3



### Текст задачи

Переписать в другой текстовый файл самую короткую и самую длинную строки.


### Алгоритм решения

Минимальная строка находится путем считывания строки и проверкой длины считаной строки, если новая строка меньше, чем текущая минимальная, то минимальная строка перезаписывается.
Максимальная строка находится схожим образом, если новая строка больше, чем текущая максимальная, то максимальная перезаписывается.
Минимальные и максимальные строки складываются в одну и записываются в новый файл.

### Алгоритм решения

После содержимых обоих файлов выводятся на экран для проверки, тк бинарный файл не доступен для чтения как файл txt

## Задание 8



### Текст задачи

Есть перечень компьютерных фирм, где может производиться закупка.
Ответить на следующие вопросы: 1) в каких фирмах закупка производилась каждым из заведений? 2) в каких фирмах закупка производилась хотя бы одним из заведений? 3) в каких фирмах ни одно из заведений не закупало компьютеры?


### Алгоритм решения

Список заполняется школами, школы заполняеются центрами, в которых они закупали технику.
Затем на основе этих данных проверяются в каких магазигах закупались все школы
после в каких магазинах закупалась хотя бы одна школа
на основе предыдущей информации находятся магазины, в которых не проводилась закупка
Результаты выводятся на экран

## Задание 9



### Текст задачи


Файл содержит текст на русском языке. Напечатать в алфавитном порядке все звонкие согласные буквы, которые входят хотя бы в одно слово.


### Алгоритм решения


Считывается строка и проверяется каждый символ на вхождение в строку звонких согласных букв, если такая имеется, то она отмечается в коллекции, как найденная в тексте
После Коллекция переводится в список и сортируется, чтобы получить все буквы в алфавитном порядке.
Результат выводится на экран

## Задание 10



### Текст задачи

Программа, которая формирует и печатает уникальный логин для каждого ученика


### Алгоритм решения


Из файла считывается количество учеников, затем считывается каждый ученик.

Если в файле уже встречалась такая фамилия, то она увеличивает в словаре количество людей с такой фамилией и выводится на экран с индивидуальным номером.
